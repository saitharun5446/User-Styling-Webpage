HTML, CSS & Java Script

HTML

HTML stands for Hyper Text Markup Language. It is called as presentation language. It is also called as formatting language. It is used to create a web page. It was developed by Tim Berners Lee in 1993.

CSS

CSS stands for Cascading Style Sheets. It is called as style sheet language. It is used to define style sheets to a web page.It was developed by Hakon Wium Lie in 1996.

Java Script

Java Script is an object oriented programming language. It is used to add dynamic behaviour to a web page. It is also used to validate html forms. It was developed by Brendan Eich in 1995.
